# AI Stock Market Dashboard

An interactive stock market intelligence dashboard built using Python and Streamlit. The project combines technical analysis indicators and news sentiment analysis to generate market insights and trading signals through a clean and user-friendly interface.

## Features

- Real-time stock market analysis
- Candlestick and line chart visualization
- RSI indicator
- SMA20 & SMA50 moving averages
- News sentiment analysis
- BUY / SELL / HOLD signal generation
- Multiple stock selection
- Custom timeframe selection
- Interactive Plotly charts

## Tech Stack

- Python
- Streamlit
- Plotly
- Pandas
- yFinance
- TextBlob
- Git & GitHub

## How to Run

Clone the repository:

```bash
git clone https://github.com/harsha-dev-op/ai-stock-market-dashboard.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the dashboard:

```bash
streamlit run app/dashboard/dashboard.py
```
